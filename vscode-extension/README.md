# monia flutter getx page template


### 使用说明

点击左下角的`monia-generate` 文字按钮，输入`pageName`，即可在`pages`目录下新建一个`flutter getx page`：

![example](example.gif)


模版来自：[monia-cli](https://github.com/xieyezi/monia-cli)

`monia-cli` 是一款同时支持 `Vue`、`React`、`Flutter` 的脚手架。
